# Release Note Ver. 2.0.7

## 新機能

### 1. 仕訳明細入力画面の機能改善

- 従来の「振替伝票」画面の名称を、より実態に即した「仕訳明細入力」に変更しました。
- 複合仕訳を入力する際、相手勘定科目を「諸口」として表示するオプション機能を追加しました。
  - この機能は、`[設定]` > `[システム設定]` > `[入力補助]` の「複合仕訳で相手勘定科目を「諸口」と表示する」から有効化できます。
  - 有効にすると、貸借片方の勘定科目のみが入力されている明細行で、空の勘定科目欄に「諸口」と表示され、入力の手間を省きます。

## 修正されたバグ

### 1. サイドバーメニューの表示不具合

- 画面遷移した際に、現在表示している画面に対応するサイドバーメニュー項目が、正しくハイライトされない不具合を修正しました。

## ドキュメント

### 1. 操作マニュアルの追加

- 各機能の操作方法や仕様を解説する、詳細なオンラインマニュアルを追加しました。
- アプリケーションのインストール方法から、日々の伝票入力、顧客管理、各種設定まで、Hieronymusを効果的に活用するための情報を網羅しています。
- マニュアルは `documents/` ディレクトリに格納されています。
